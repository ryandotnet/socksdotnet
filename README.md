# socksdotnet
A basic proxy with SOCKS4 / SOCKS4a / SOCKS5 protocol support. Written in .NET 7.0 

# Features:
1. CONNECT support
2. DNS resolution (through proxy)
3. Username & Password Authentication
4. External configuration via. json file
